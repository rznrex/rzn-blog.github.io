## 前言

Hugo 是一个用 Go 语言编写的静态网站生成器，以**极快的构建速度**和**简单的安装过程**著称[reference:0][reference:1]。与很多依赖复杂运行环境的静态生成器不同，Hugo 仅需要一个二进制文件即可运行[reference:2]。本教程将一步步带你完成在 iOS 系统上安装 Hugo、创建博客，并最终通过 GitHub Actions 自动部署到 GitHub Pages 的全过程[reference:3]。
